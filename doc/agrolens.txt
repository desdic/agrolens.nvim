================================================================================
INTRODUCTION                                                     *agrolens.nvim*

Its an extention for telescope that runs pre-defined (or custom) tree-sitter
queries on a buffer (or all buffers) and gives a quick view via telescope

M.generate_new_finder({opts})                        *M.generate_new_finder()*
    Generate a new finder for telescope


    Parameters: ~
        {opts} (table)  options

    Fields: ~
        {entry_maker}            (function)  function(line: string) => table
                                             (Optional)
        {cwd}                    (string)    current root directory (Optional)
        {previewer}              (function)  function(line: string) => table
                                             (Optional)
        {sorter}                 (function)  function(line: string) => table
                                             (Optional)
        {same_type}              (bool)      if buffers should be of same
                                             filetype (default false)
        {include_hidden_buffers} (bool)      if hidden buffers should be
                                             included (default false)
        {disable_indentation}    (bool)      if true it strips the whitespaces
                                             (default false)
        {debug}                  (bool)      enable debugging (default false)
        {matches}                (table)     key/value pair for matching
                                             variable names
        {aliases}                (table)     key/value pair for aliases for
                                             longer lists of queries (value is
                                             comma seperated)

M.generate({opts})                                              *M.generate()*
    Generate a query from cursor position Example usage:
    >
    require("agrolens").generate({in_buffer=false})
<


    Parameters: ~
        {opts} (table)  options

    Fields: ~
        {register}          (char)     register to copy to. Default ""
        {in_buffer}         (boolean)  create a buffer with query, default
                                       true
        {in_register}       (boolean)  copy query to register (clipboard),
                                       default true
        {full_document}     (boolean)  create query for full document/tree,
                                       default false
        {all_captures}      (boolean)  create a capture group for every
                                       possible combination, default false
        {include_root_node} (boolean)  include root node, default false
